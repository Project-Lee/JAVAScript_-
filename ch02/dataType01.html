자바스크립트의 자료형(Data Type)

1. 기본형
(1) 숫자형(number) : 따옴표 없이 표기한 숫자
- 정수와 실수를 구분하지 않는다.(int, float 이런거없음)
(2) 문자형 (String) : 값을 따옴표로 써서 표기한 문자 "내용~~ "
(3) 논리형 : 값을 true,false 로 표기하는 자료형
(4) undefined형 : 선언하고, 초기화 되지 않은 자료형 = var a; var name;
2. 복합형
(1) array형 (배열) 하나의 변수에 여러 개의 값을 저장하는 자료형
(2) object형 (객체) : 함수와 속성이 포함된 자료형
